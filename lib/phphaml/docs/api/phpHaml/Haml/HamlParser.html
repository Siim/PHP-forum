Haml parser.

Haml is templating language. It is very simple and clean. Example Haml code 

  1.  !!! 1.1
    2. 

  2.  %html
    3. 

  3.    %head
    4. 

  4.      %title= $title ? $title : 'none'
    5. 

  5.      %link{ :rel => 'stylesheet', :type => 'text/css', :href => "$uri/tpl/$theme.css" }
    6. 

  6.    %body
    7. 

  7.      #header
    8. 

  8.        %h1.sitename example.com
    9. 

  9.      #content
    10. 

  10.        / Table with models
    11. 

  11.        %table.config.list
    12. 

  12.          %tr
    13. 

  13.            %th ID
    14. 

  14.            %th Name
    15. 

  15.            %th Value
    16. 

  16.          - foreach ($models as $model)
    17. 

  17.            %tr[$model]
    18. 

  18.              %td= $model->ID
    19. 

  19.              %td= $model->name
    20. 

  20.              %td= $model->value
    21. 

  21.      #footer
    22. 

  22.        %address.author Random Hacker
    23.
Comparing to original Haml language I added:
  • Support to translations - use '$' 
    1.  %strong$ Log in
      2.
  • Including support ('!!') and level changing ('?') 
    1.  !! html
      2. 

    2.  !! page.header?2
      3. 

    3.  %p?3
      4. 

    4.    Foo bar
      5. 

    5.  !! page.footer?2
      6.
